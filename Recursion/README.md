# 🔁 Recursion

This folder demonstrates the use of recursion to solve mathematical and logical problems.  
Each program focuses on breaking problems into smaller subproblems until a base condition is reached.

## 💡 Concepts Covered

- Recursive thinking and problem decomposition
- Base case and recursive case design
- Mathematical recursion problems
- Comparison of recursive vs iterative approaches

These examples help build strong problem-solving skills required for advanced algorithms.
