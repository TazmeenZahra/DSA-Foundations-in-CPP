# 🔎 Searching and Sorting Basics

This folder contains implementations of fundamental searching and sorting algorithms written in C++.  
These programs focus on understanding how data can be organized efficiently and how search operations can be optimized.

## 💡 Concepts Covered

- Bubble Sort
- Selection Sort
- Binary Search
- Working with arrays without built-in functions
- Performance awareness and algorithmic thinking

These implementations build the foundation for advanced data structures and algorithm optimization.
